Unit Summaries
Unit 1

Summary

In the first unit, we learned about websites and web pages and how to add content, such as images, links, and text. We also learned many tags for HTML, such as "p" for a paragraph, "h1" for a large heading, etc. We learned about binary numbers and how to convert them into a decimal value and vice versa. More importantly, we learned how to use CSS, specifically in-line, internal, and external CSS.

Unit 2

Summary

In unit 2, we learned a lot about the internet, specifically how data is transferred, IP addresses, protocols, computing networks, etc. We learned the process of data transfer, all the steps required to do it, and how to make it secure.

Unit 3

Summary

In this unit, we learned about apps and their components. An app is made of its user interface and inputs/outputs. Inputs are actions such as taps and clicks, and output is a change to what the user sees in response, such as the screen changes. We learned how to make apps and utilize them to make them function properly.

Unit 4

Summary

This unit taught us about variables in JavaScript, which are named and contain a value, either a string or a number. We learned how to utilize these variables in our app to add extra functionality to them and display them on the screen. We learned about events and how we can use inputs to cause something to change in response. (output). We also learned about logical and comparison operators, both of which can be used in a conditional statement, and assignment operators which assign a new value to a previously declared variable.
