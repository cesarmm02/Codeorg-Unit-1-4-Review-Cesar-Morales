Check for Understanding

Programming Languages vs. Natural Languages

What is the difference between a programming language and natural (every-day) language?

Answer
A programming language is more specific and you cannot assume things, unlike natural language. The computer does not know what to do unless you tell it exactly.

Check for Understanding

Sequential vs. Event Driven Programs

What is the difference between a sequential program and an event-driven program?

Answer
A sequential program goes in a specified order while the event-driven program only triggers something after a certain event, such as a click, or any input
